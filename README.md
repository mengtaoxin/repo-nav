# Repo Nav

Starter workspace for a Next.js + TypeScript + shadcn/ui app.

## Getting started

- Start the dev server: `npm run dev`
- Build for production: `npm run build`
- Run lint checks: `npm run lint`
- Run tests: `npm run test`

## UI components

Add shadcn/ui components on demand:

- `npx shadcn@latest add button`

## Notes

- Update the landing content in `src/app/page.tsx`.
- Shared layout is in `src/app/layout.tsx`.
